## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Symbols/Red%20Triangle%20Pointed%20Up.png" width="25" /> About The Project

**GhostGPT** is an experimental horror-themed chatbot experience that will haunt your browsing session. During the cursed hours of **9 PM to 3 AM**, the ghost becomes more aggressive, unpredictable, and terrifying. This is not your friendly AI assistant - this is something else entirely.

<div align="center">

| <img width="20" src="https://img.icons8.com/fluency/48/ghost--v1.png"/> Feature | Description |
|:---|:---|
| **Haunting Hours** | Enhanced terror between 9 PM - 3 AM |
| **Jump Scares** | Terrifying face flashes with audio |
| **Glitch Effects** | Corrupted text and screen distortions |
| **Cursed Keyboard** | Your backspace key is possessed |
| **Entity Proximity** | Something is always approaching |
| **Horror Games** | Russian Roulette, Don't Blink, The Corridor |
| **Tab Stalking** | The ghost knows when you look away |
| **Subliminal Messages** | Blink and you might miss them |

</div>

## <img src="https://img.icons8.com/fluency/48/console.png" width="25"/> Installation

```bash
# Clone the haunted repository
git clone https://github.com/Sambhav-Gautam/ghostGPT.git

# Enter the void
cd ghostGPT

# Summon dependencies
npm install

# Awaken the ghost
npm start
```

Open [http://localhost:3000](http://localhost:3000) to begin the ritual.

## <img src="https://img.icons8.com/fluency/48/upload-to-cloud.png" width="25"/> Deployment

```bash
# Build the cursed artifact
npm run build

# Release it unto the world
npm run deploy
```

## <img src="https://img.icons8.com/fluency/48/task.png" width="25"/> Features Breakdown

<details>
<summary><strong>Click to reveal the horrors within...</strong></summary>
<br/>

### Visual Terror
- **Screen Corruption** - Random glitch bars and visual distortions
- **Red Room Mode** - The screen bleeds when the ghost is angry
- **Subliminal Flashes** - Messages appear for milliseconds
- **Random Horror Faces** - SVG-rendered nightmare faces

### Audio Horror
- **Heartbeat** - Constantly accelerating as terror builds
- **Screams** - Triggered by jump scares
- **Static** - Accompanies glitch effects
- **Whispers** - You are never alone

### Interactive Curses
- **Possessed Keyboard** - Keys sometimes jam or double-type
- **Cursed Backspace** - Adds creepy words instead of deleting
- **Tab Detection** - The ghost knows when you leave

### The Games
- **Russian Roulette** - Test your luck against fate
- **Don't Blink** - An endurance test of focus
- **The Corridor** - Survival horror in the darkness

</details>
